1. Reverse a String using Stack
Description: This program reverses a given string using a stack.
Input Example: DATA
Output Example: ATAD
Key Concept: Uses stack's LIFO (Last In, First Out) property to reverse characters.
2. Balanced Parentheses
Description: Checks whether the parentheses in a mathematical expression are balanced.
Input Example: (A+B)*(C-D)In this assignment, you will learn how stacks and queues work by writing C programs. You will reverse a string using a stack, check if the brackets in an expression are balanced, and find the next bigger number for each element in an array. You will also create a printer queue to see how documents are printed in order, and make a circular queue to manage elements efficiently. These programs will help you understand how stacks and queues are used in real life and give you practice in writing C program
Output Example: Balanced Expression
Key Concept: Uses a stack to push opening parentheses and pop them when matching closing parentheses appear.
3. Next Greater Element
Description: Finds the next greater element for each element in an array using a stack.
Input Example: 4 5 2 10 8
Output Example:
4 → 5
5 → 10
2 → 10
10 → -1
8 → -1
Key Concept: Uses stack to efficiently find the next larger element for each array item.
4. Printer Queue Simulation
Description: Simulates a printer queue system using queue operations.
Operations Supported:
Add document to queue
Print document
Display pending documents
Key Concept: Uses queue (FIFO - First In, First Out) to simulate real-world printing order.
5. Circular Queue Basic Operations
Description: Implements a circular queue using an array with menu-driven operations.
Operations Supported:
Enqueue – Insert an element
Dequeue – Remove an element
Peek – Display the front element
Display – Show all elements
Key Concept: Circular queue avoids wasted space and efficiently uses array storage by connecting the end back to the start.
